ทำ AJAX ของ Drop Down Menu ที่ไว้เลือกจังหวัด อำเภอ ตำบล ของไทย

*วิธีการติดตั้ง*

1. ให้ ไปที่ Control Panel ของเว็บไซต์ของคุณ สร้างฐานข้อมูลชื่อ thailand_province (จะเป็นชื่ออื่นก็ได้นะ)
2. พอคลาย ZIP ออกมา จะพบไฟล์ครับ
3. ใช้โปรแกรม Notepad (อยู่ที่ Start -> All Programs -> Accessories -> Notepad) เปิด File ชื่อ dbconfig.php ครับ
4. ไปแก้ไข Code ภายในนี้ครับ (เป็นค่าที่เกี่ยวกับการเชื่อมต่อฐานข้อมูล ตั้งให้สอดคล้องกับที่สร้างไว้ในข้อ 1 นะ)
5. แก้เสร็จแล้วก็ Save ครับ (ไปที่ File --> Save นะ)
6. ทีนี้ให้คุณ Upload ไฟล์ เข้าไปบน SERVER ของเว็บไซต์คุณเลย
7. ให้เข้าไปยัง phpMyAdmin ของ HOST คุณครับ (ไปดูที่ Control Panel ของ Host คุณ มันจะมี Link ไปอยู่น่ะครับ)
8. ให้เลือกไปยังฐานข้อมูลที่คุณได้สร้างไว้ตะกี้น่ะ
9. ให้เลือกไปที่ Import -> กด Browse แล้วเลือกไฟล์ location.sql ครับ เสร็จแล้วกดปุ่ม ลงมือ เลย (ควรเลือก character set เป็น utf8)
10. ถ้าทำสำเร็จก็จะเห็นรายการ จังหวัด อำเภอ ตำบล ในตาราง province , amphur , district ครับ
11. ก็สามารถใช้งานได้เลยครับ ( ดู Sourcecode ของตัวแบบฟอร์มในไฟล์ test.html ครับ เป็น HTML & Javascript )
12. ส่วน Script ในการรับข้อมูลจากแบบฟอร์ม พร้อมแสดงค่าที่เลือกให้ดูที่ไฟล์ send_data.php ครับ มี Sourcecode ตัวอย่าง
13. เป็นอันเสร็จสิ้นขั้นตอนการติดตั้งครับ สามารถใช้งานได้เลย
14. ใครมีข้อสงสัยอะไรก็อีเมล์มาถามกันได้นะครับที่ au_cty@hotmail.com ครับ
